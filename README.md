# FUTURE_UX_02

## Task 2 - Mobile App UI for Appointment Booking

### Overview
This project was created as part of the UI/UX Design Internship at Future Interns.

The objective was to design a mobile application UI for an appointment booking system that can be used by service-based businesses such as salons, clinics, consultants, and fitness trainers.

---

## Project Features

- Splash Screen
- Login / Sign Up Screen
- Home Dashboard
- Service Selection Screen
- Date & Time Booking Interface
- Appointment Confirmation Screen

---

## Design Goals

- Create a simple and user-friendly booking experience.
- Improve appointment scheduling efficiency.
- Maintain a clean and modern mobile UI.
- Ensure smooth navigation between screens.

---

## Design Process

### Research
Studied modern appointment booking applications and mobile UI patterns.

### Wireframing
Created low-fidelity layouts to define the user flow.

### UI Design
Designed high-fidelity screens with a consistent color palette, typography, and visual hierarchy.

### Prototyping
Connected screens to simulate the booking journey.
